## Subprojects

- `calibration/` — Empirical calibration analysis of resolved Polymarket binary markets. Measures whether market prices correspond to real-world resolution frequencies using pre-resolution CLOB prices. Self-contained pipeline (fetch → preprocess → analyze → visualize). Merged from the archived `polymarket-calibration` repo. See `calibration/README.md` for usage.
