# Polymarket 조합 전략 TOP 10

**분석 기준:** 실제 코드 기반 시뮬레이션 + 시너지 분석 + 상관관계 테스트
**업데이트:** 2026-02-28
**평가 요소:** 시장 커버리지, 리스크 분산, 엣지 상관관계, 실행 가능성, 합의 신호 강도

---

## 왜 조합 전략인가?

단일 전략은 특정 시장 유형에만 반응한다. 조합 전략은:
- **시장 커버리지 확대:** 한 전략이 놓치는 기회를 다른 전략이 포착
- **리스크 분산:** 서로 다른 엣지 소스로 한 전략 실패 시 다른 전략이 보완
- **합의 신호:** 여러 전략이 같은 방향을 가리키면 확신도 급상승
- **수익 안정화:** 변동성 감소, 최대 낙폭(MDD) 축소

### 시뮬레이션 방법

30개 합성 시장(9개 카테고리) 대상으로 15개 조합을 테스트. 평가 지표:
- **커버리지:** 전체 시장 중 신호 발생 비율
- **상보성:** 전략 간 중복 없이 서로 다른 시장을 커버하는 정도
- **리스크 다양성:** 엣지 유형과 리스크 스타일 분포
- **합의 강도:** 동일 시장에 복수 전략이 일치할 때의 확신도

## 1위: 모멘텀 + 감성 + 수익 조합

**구성:** S39 (거래량 모멘텀) + S45 (트위터 감성 역투자) + S49 (스테이블코인 수익)

### 시너지 분석

| 지표 | 값 | 해석 |
|------|-----|------|
| 시장 커버리지 | 63.3% (19/30) | **전체 1위** -- 가장 많은 기회 포착 |
| 상보성 | 1.000 (완벽) | 세 전략 간 중복 **제로** |
| 리스크 스타일 | 3개 (모멘텀/역발상/패시브) | 완벽한 분산 |
| 엣지 유형 | 3개 (모멘텀/감성/수익) | 서로 다른 알파 소스 |
| 카테고리 | 8개 | 거의 모든 시장 유형 커버 |

### 왜 이 조합이 최강인가

```
S39 (모멘텀): 거래량 3배 이상 급등 시장 탐지 → 16개 시장 신호
    ↕ 겹치지 않음
S45 (감성): 트위터에서 80%+ 한 방향 감성 → 역방향 베팅
    ↕ 겹치지 않음
S49 (수익): 90%+ 고확률 시장에서 안정 수익 수확
```

- **S39**가 뜨거운 시장(고거래량)을 찾으면 → 단기 방향성 포착
- **S45**가 감성 과열 시장을 찾으면 → 중기 평균회귀 포착
- **S49**가 확실한 시장을 찾으면 → 무위험 수익으로 기본 수익 확보
- **세 전략이 절대 같은 시장에 베팅하지 않아** 포지션 충돌 없음

### 실전 운용법

```
전체 자본: $10,000 기준

S39 (모멘텀): 40% = $4,000
  - 고거래량 시장 빠른 진입/퇴출
  - 포지션당 $200-$400
  - 하루 2-5건 거래

S45 (감성 역투자): 35% = $3,500
  - 트위터 과열 역베팅
  - 포지션당 $300-$500
  - 주 3-5건 거래

S49 (수익 수확): 25% = $2,500
  - 90%+ 확실한 시장 보유
  - 포지션당 $500-$1,000
  - 월 2-3건 거래 (장기 보유)
```

### 필요 데이터

- 거래량 데이터 (Polymarket API -- 기본 제공)
- 트위터 감성 데이터 (선택사항, 없어도 S39+S49만으로 운용 가능)

## 2위: 역발상 + 모멘텀 조합

**구성:** S01 (바보 반대편 베팅) + S39 (거래량 모멘텀)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 시장 커버리지 | 53.3% (16/30) |
| 상보성 | 0.938 |
| 합의 시장 | 1개 (확신도 0.625, 엣지 13.5%) |
| 카테고리 | 7개 |

### 왜 이 조합이 강한가

```
시장 과열 감지 → 두 가지 경로:

경로 1 (S39): 거래량 폭발 → 모멘텀 방향 베팅
경로 2 (S01): 감정 과열 → 반대 방향 베팅

두 전략이 같은 시장을 가리키면?
→ "거래량은 폭발했지만 감정적으로 과열" = 매우 강한 역발상 신호
→ 합의 엣지 13.5% (평균보다 3배 높음)
```

- **상호 검증 효과:** S39가 "거래량 급등"을 포착하고 S01이 "감정 과열"을 확인하면, 둘이 합쳐서 더 강한 신호 생성
- **단독 신호도 유효:** 각자 독립적으로도 충분한 엣지 보유
- **외부 데이터 최소:** 기저율 + 뉴스 API만 필요 (뉴스 없어도 작동)

### 실전 운용법

```
전체 자본: $5,000 기준

S01 (역발상): 60% = $3,000
  - 대형 이벤트 후 감정 과열 시장
  - 고확신 시장에 집중

S39 (모멘텀): 40% = $2,000
  - 거래량 급등 시장 빠른 대응
  - 분산 투자

합의 신호 발생 시: 양쪽에서 추가 배분 (최대 2배)
```

## 3위: 풀 역발상 스택

**구성:** S01 (바보 반대편) + S03 (아무 일 없음) + S10 (YES 편향) + S15 (뉴스 평균회귀)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 시장 커버리지 | 43.3% (13/30) |
| 합의 시장 | **4개** (확신도 0.617, 엣지 **18.5%**) |
| 카테고리 | 7개 |
| 합의 엣지 | **전체 조합 중 최고** |

### 왜 이 조합이 강한가

네 가지 전략이 모두 "시장이 과대반응한다"는 같은 철학을 공유하지만, **각각 다른 신호를 감지**:

```
같은 시장을 4개 전략이 동시에 가리킬 때:

S01: "감정적으로 과열됨" ✓
S03: "극적 사건은 안 일어남" ✓
S10: "YES 편향이 작용 중" ✓
S15: "뉴스 과잉반응 후 회귀 예상" ✓

→ 4중 확인 = 엣지 18.5% (단일 전략 평균의 4배)
→ 이런 시장에 집중 배팅하면 승률 극대화
```

### 실전 운용법

```
자본 배분 규칙:
- 1개 전략만 신호: 표준 포지션 ($200)
- 2개 전략 합의: 1.5배 ($300)
- 3개 전략 합의: 2배 ($400)
- 4개 전략 합의: 3배 ($600) ← 최대 확신 베팅
```

## 4위: 기저율 트리오

**구성:** S01 (바보 반대편) + S03 (아무 일 없음) + S10 (YES 편향)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 시장 커버리지 | 40.0% (12/30) |
| 상보성 | 0.944 |
| 합의 시장 | 2개 (확신도 0.637, 엣지 **19.8%**) |

### 핵심 시너지

3위(풀 스택)의 경량화 버전. 뉴스 API 없이도 작동하며, **합의 시 엣지가 오히려 더 높음(19.8% vs 18.5%)**. S15(뉴스 평균회귀)가 빠지면서 불필요한 노이즈가 줄어드는 효과.

```
추천 상황:
- 뉴스 API 없을 때
- 간단한 운용을 원할 때
- 정치/지정학 시장 집중 시
```

## 5위: 차익거래 풀스택

**구성:** S04 (크로스플랫폼) + S05 (NegRisk) + S02 (날씨 NOAA)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 시장 커버리지 | 26.7% (8/30) |
| 상보성 | 1.000 (완벽) |
| 엣지 유형 | 3개 (크로스플랫폼/구조적/데이터) |
| 리스크 | **전체 조합 중 최저** |

### 왜 이 조합이 특별한가

**세 전략 모두 "수학적 확실성"에 기반한 차익거래:**

```
S04: Polymarket YES + Kalshi NO < $1.00 → 무위험 수익
S05: 다중결과 YES 합계 > $1.00 → 무위험 수익
S02: NOAA 과학 데이터 vs 직감 가격 → 정보 우위 수익

세 전략의 공통점: "수학/과학이 맞고 시장이 틀리다"
세 전략의 차이점: 전혀 다른 시장 유형 (이종/다중결과/날씨)
```

### 실전 운용법

```
자본 배분:
S05 (NegRisk): 50% -- 가장 높은 자본 효율성 (29배)
S04 (크로스플랫폼): 30% -- 양 플랫폼 분산 필요
S02 (날씨): 20% -- 마이크로 베팅으로 분산

필요 인프라:
- Kalshi 계좌 + API
- NOAA API (무료)
- 자동화 봇 (S04, S05 필수)
```

|------|-----|

### 핵심 시너지

**"안전 제일" 조합.** 세 전략 모두 높은 확률에 베팅:

```

S03: 70% 기저율 → NO에 체계적 베팅

합의 시: 확신도 0.825 (거의 확실)
```

- **보수적 투자자에게 최적**
- 최대 낙폭(MDD) 최소화
- 연 20-40% 안정 수익 목표

## 7위: 메타 포트폴리오 조합

**구성:** S19 (켈리 사이징) + S28 (포트폴리오 에이전트) + S100 (메타 앙상블)

| 지표 | 값 |
|------|-----|
| 커버리지 | 33.3% (10/30) |
| 상보성 | 1.000 |
| 외부 데이터 | **불필요** (자체 완결) |

### 핵심 시너지

```
  → S100이 전체 포트폴리오의 가중 앙상블 조정

핵심 가치: 다른 전략들의 "위에 올리는" 메타 레이어
```

- 예: 4위(기저율 트리오) + 7위(메타) = 최적화된 기저율 포트폴리오

## 9위: HFT + 패시브 조합

**구성:** S06 (BTC 레이턴시) + S12 (고확률 수확) + S49 (스테이블코인 수익)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 커버리지 | 16.7% (5/30) |

### 핵심 시너지

```

  ↕ 완전 무상관
S12 + S49: 장기 (일~주 단위) 저위험 안정 수익

S06이 실패해도 S12+S49가 기본 수익 보장
S06이 성공하면 전체 수익률 폭발
```

- **인프라 투자 가능한 고급 트레이더 전용**
- S06 없이 S12+S49만으로도 유효

## 10위: 뉴스 + 역발상 조합

**구성:** S15 (뉴스 평균회귀) + S45 (트위터 감성 역투자) + S01 (바보 반대편)

### 시너지 분석

| 지표 | 값 |
|------|-----|
| 커버리지 | 16.7% (5/30) |
| 상보성 | 1.000 |
| 엣지 유형 | 3개 (평균회귀/감성/역발상) |

### 핵심 시너지

**"이벤트 반응 전문" 조합:**

```
대형 뉴스 발생 시:
  S15: "24시간 내 15%+ 이동 → 50% 회귀 예상"
  S45: "트위터 80%+ 한 방향 → 역방향 베팅"
  S01: "감정 과열 → 기저율로 회귀"

세 전략 모두 "과잉반응 후 회귀"를 포착하지만:
  S15 → 가격 움직임 기반
  S45 → 소셜 미디어 감성 기반
```

## 핵심 인사이트

### 1. 커버리지가 수익을 결정한다
단일 전략 최대 커버리지: 53% (S39)
조합 전략 최대 커버리지: **63.3%** (1위 조합)
→ 더 많은 기회 = 더 많은 수익 기회

### 2. 합의 신호는 금이다
복수 전략이 같은 시장에서 같은 방향을 가리킬 때:
- 평균 엣지 **18.5%** (단일 전략 평균 5%의 3.7배)
- 평균 확신도 **0.825** (단일 전략 평균 0.6의 1.4배)

### 3. 상보성이 핵심이다
같은 유형 전략을 3개 쓰는 것보다, **다른 유형 전략을 3개** 쓰는 것이 항상 우월:
- 같은 유형 3개: 커버리지 40%, 중복 높음
- 다른 유형 3개: 커버리지 63%, 중복 제로

### 4. 메타 전략은 앰프다
S19/S28/S100은 단독으로 약하지만, 다른 신호 생성 조합 위에 올리면 **자본 배분 최적화**로 수익률 15-25% 추가 개선.
